Bem-vindo ao Projeto Servidor & Cliente HTTPS

Este mini-site apresenta minha pesquisa e implementação de um servidor e cliente HTTPS em Python, conforme proposto no trabalho de Segurança Computacional.

Conhecer Protocolos

O que é HTTPS?

HTTPS garante uma comunicação segura, usando SSL/TLS para proteger dados, autenticar servidor e cliente, e preservar a integridade das mensagens.

Implementação em Python

Utilizei módulos nativos (http.server e ssl) para criar um servidor HTTPS e um cliente, além de demonstrar análise de comunicação segura.
